Task to generate a Flutter UI.

Views can be constructed using a Klutter Kompose DSL, being one of:

  • JetlagUI

  • KlutterUI

  • NotSwiftUI

KlutterUI is the default DSL which resembles Flutter the most. JetlagUI is a DSL which is inspired by Jetpack Compose. NotSwiftUI is a DSL which is inspired by SwiftUI.

Each DSL returns the same set of Kompose objects which are used to generate the Flutter UI.
